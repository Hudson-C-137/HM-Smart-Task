# HM-Smart-Task

📅 HM Smart Task (Offline PWA)
Uma ferramenta de produtividade minimalista e de alta performance, desenvolvida para funcionar 100% offline. Inspirada no ecossistema Apple (iOS/macOS), ela combina a fluidez de um aplicativo nativo com a praticidade de uma página HTML única.

✨ Funcionalidades Principais
📱 Design Adaptável (PWA): Interface responsiva que se comporta como um aplicativo nativo tanto em PCs quanto em dispositivos móveis.

🗓️ Calendário Inteligente: Filtre suas tarefas por dia ou mês. Ao abrir, o app identifica automaticamente a data atual.

🗂️ Cards em Slide: Visualização moderna das tarefas do dia com suporte a gestos de deslize em telas touch.

✅ Gestão Dinâmica: * Check-list interativo dentro de cada card.

Botões de ação rápida para "Concluir Tudo" ou "Resetar para Pendente".

Indicadores Visuais: Contorno dinâmico (Verde para concluído, Vermelho para pendente) e barra de progresso em tempo real.

💾 Backup e Segurança:

Exportação de dados em formato JSON.

Importação de backups para sincronia entre dispositivos.

Opção de "Restaurar Padrão de Fábrica" para limpeza total.

🔒 Privacidade Total: Seus dados nunca saem do seu dispositivo. Tudo é armazenado localmente via localStorage.

🛠️ Tecnologias Utilizadas
HTML5/CSS3 (Estrutura e Estilização)

Tailwind CSS (Framework utilitário para design responsivo)

JavaScript Vanilla (Lógica de persistência e manipulação de DOM)

JSON (Estrutura de dados para backup)

PWA Manifest (Para instalação direta no Android/iOS/Windows)
